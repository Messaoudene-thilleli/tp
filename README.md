# Principes de programmation

Ce dépôt regroupe les différents TP réalisés durant les séances de principes de programmation.

## Objectif

Les différents travaux réalisés permettent de mettre en pratique :

- la création et l’utilisation d’API REST ;
- la consommation d’API avec PHP ;
- l’utilisation du framework Flask ;
- la conteneurisation avec Docker ;
- la communication entre services web.

---

# Organisation

Le dépôt contient plusieurs dossiers correspondant aux différents TP :

- **TP1**  
  Introduction et premiers exercices.

- **TP2 — Client PHP et API REST**  
  Application PHP permettant d’afficher des étudiants à partir d’une API REST.

- **TP4**  
  Suite des manipulations autour des services web.

- **TP5 — Docker**  
  Conteneurisation d’une API Flask avec Docker.

- **TP6**  
  Publication et manipulation d’images Docker.

- **TP7**  
  Manipulations complémentaires autour de Docker.
